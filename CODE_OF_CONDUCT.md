# 貢献者行動規範 (Contributor Code of Conduct)

## 私たちの誓い (Our Pledge)
私たちは、オープンで心地よい環境を育むため、貢献者やメンテナーとして、年齢、体の大きさ、障碍、民族性、性自認およびジェンダー表現、経験レベル、教育、社会経済的地位、国籍、個人の容姿、人種、宗教、性的同一性および指向に関わらず、誰もが嫌がらせを受けることなくプロジェクトやコミュニティに参加できることを誓います。

## 私たちの基準 (Our Standards)
前向きな環境作りに貢献するふるまいの例：

* 歓迎的で誰をも受け入れる言葉を使う

* 多様な視点や経験を尊重する

* 建設的な批判を品位をもって受け入れる

* コミュニティにとって何が最善であるかに焦点を当てる

* 他のコミュニティメンバーに共感を示す

参加者による容認できないふるまいの例：

* 性的な意味を含む言葉や画像、相手の望まない性的な関心や接近

* あおり、侮辱的・軽蔑的なコメント、個人攻撃や政治攻撃

* 公的または私的な嫌がらせ

* 住所、メールアドレスなど、他者の個人情報を本人の明確な許可なく公開すること

* その他、専門的な場にふさわしくないと合理的に考えられる行為

## 私たちの責任 (Our Responsibilities)

プロジェクトのメンテナーは、許容されるふるまいの基準を明確にする責任があります。また、許容できないふるまいに対して、適切かつ公正な是正措置を講じる責任があります。

プロジェクトのメンテナーは、この行動規範に沿わないコメント、コミット、コード、wiki編集、issue、その他の貢献を削除、編集、拒否する権利と責任を有します。また、不適切、脅迫的、攻撃的、または有害であると見なされるその他のふるまいをした貢献者を一時的または恒久的に追放する権利を有します。

## 適用範囲 (Scope)

この行動規範は、個人がプロジェクトまたはそのコミュニティを代表する場合、プロジェクトスペースと公共スペースの両方に適用されます。プロジェクトやコミュニティを代表する例には、プロジェクトの公式メールアドレスの使用、公式ソーシャルメディアアカウントを介した投稿、オンラインまたはオフラインのイベントで指定された代表者として行動することなどが含まれます。プロジェクトの代表は、プロジェクトのメンテナーによってさらに定義され、明確化されることがあります。

# 強制 (Enforcement)

虐待的、嫌がらせ、またはその他容認できないふるまいに遭遇した場合、またはその他の懸念がある場合は、[ここにプロジェクトの連絡先を挿入] までご連絡ください。すべての苦情はレビューされ、調査され、その状況に応じて必要かつ適切であると見なされる対応がなされます。プロジェクトチームは、事象の報告者に関して守秘義務を負います。具体的な強制方針の詳細は、別途掲載されることがあります。

この行動規範に従わない、または強制しないプロジェクトメンテナーは、プロジェクトリーダーの他のメンバーの判断により、一時的または恒久的な影響を受ける可能性があります。
